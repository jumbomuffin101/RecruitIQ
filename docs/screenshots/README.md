# Screenshot Capture Plan

Capture these screens from a populated RecruitIQ workspace using synthetic or user-created test data:

1. `landing.png` - public landing page at desktop width.
2. `dashboard.png` - dashboard with Action Center and pipeline health metrics.
3. `candidate-evaluation.png` - candidate evaluation breakdown, requirement evidence drawer, and recommendation.
4. `compare.png` - job-specific candidate ranking and requirement matrix.
5. `pipeline.png` - pipeline filtered to a single role, with application-specific stages visible.
6. `interview-scorecard.png` - completed scorecard and feedback validation.

Use real application captures only. Do not replace these files with generated mockups or synthetic screenshots.
